The Boxes below are an example of responsive layout. In desktop all the way to 1024px they behave as 2 column components. Between 1024 to 768px they behave as a 1 column component, below 768px until 320px they again span to a 2 column layout

They achieve this using the right set of classes (col-# for desktop, col-md-# for tablets and col-sm-# for mobile phones)
